# Whole Picture — app

The SvelteKit app for **MyWealthProjection** — a chart-first retirement drawdown model.

## Docs
- **Product vision** (the north star): [`docs/VISION.md`](./docs/VISION.md)
- **Where the build is, design decisions, backlog**: [`docs/STATUS.md`](./docs/STATUS.md)

## Run

```sh
npm install
npm run dev        # http://localhost:5173
```

## Other scripts

```sh
npm run build      # production build
npm run preview    # preview the production build
npm run check      # svelte-check (types)
npx vitest run     # unit tests (domain layer)
```
